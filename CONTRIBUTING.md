# Contributing to RiftChanger

Thank you for your interest in contributing to RiftChanger! 🎨

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/RiftChanger.git`
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Commit with a descriptive message: `git commit -m "feat: add your feature"`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

```bash
npm install
npm run electron:dev
```

## Code Style

- Use TypeScript for all new code
- Follow existing naming conventions
- Write meaningful commit messages using [Conventional Commits](https://www.conventionalcommits.org/)
