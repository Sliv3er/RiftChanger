# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, email **ghodhbaneayoub@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond within 48 hours and work with you to resolve the issue.
